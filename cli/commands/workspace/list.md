# Command: workspace list

List all the workspaces.

## Synopsis

```
noodl-cli workspace list [--username <username>] [--password <password>]
```

### Configuration

#### `username`

- Default: undefined
- Type: String

Noodl username.

#### `password`

- Default: undefined
- Type: String

Noodl password.
