# Command: project list

List all the projects in a workspace.

## Synopsis

```
noodl-cli project list <workspace-id> [--username <username>] [--password <password>]
```

### Configuration

#### `username`

- Default: undefined
- Type: String

Noodl username.

#### `password`

- Default: undefined
- Type: String

Noodl password.
